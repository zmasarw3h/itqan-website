# ITQAN Daily Check-In — Emergency Build Brief

## Purpose
Build a temporary one-masjid check-in system while Canvas is unavailable.

## Core Goal
Students can mark whether they completed their work for the day.
Admins can view student completion by date/week.

## Roles
- Student
- Admin

## In Scope
- Student login
- Student daily completion check-in
- Student check-in history
- Admin dashboard
- Admin can view all students and check-ins
- Admin can filter by date/status/student
- Admin can export CSV
- Admin can manually correct check-ins

## Out of Scope
- Teacher role
- Weekly plan submission
- Saturday scoring
- Sadaqa calculation
- Announcements
- Multi-masjid support
- Payment
- Quran ayah selector
- Booking/scheduling

## Constraints
- One masjid only
- Mobile-friendly
- Simple and fast to build
- Minimal student data
- Must prevent duplicate check-ins for the same student/date
