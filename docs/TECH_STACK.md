# Tech Stack

- Framework: Next.js
- Database/Auth: Supabase
- Hosting: Vercel
- Styling: Tailwind
- Export: CSV download

## Required Pages
- /login
- /student/check-in
- /student/history
- /admin
- /admin/students/[id]

## Environment Variables
- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY
- SUPABASE_SERVICE_ROLE_KEY, server-side only if needed
